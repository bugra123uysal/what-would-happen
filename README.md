Features

Calculates the annual CAGR (Compound Annual Growth Rate) for selected stocks using their price data between 2020–2025.

Shows how a 1000 USD investment would grow based on each stock’s annual compounded return.

Plots 5-year price charts for the selected stocks to visualize long-term performance.

Displays key stock fundamentals such as Market Cap, PE Ratio, EPS, Beta, and 52-week High/Low using bar charts.

Computes the correlations between different stocks to identify relationships and similarities.
